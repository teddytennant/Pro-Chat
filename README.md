## Getting Started

1. Install dependencies: `npm install`
2. Start the development server: `npm run dev`
3. Open http://localhost:5173 in your browser
4. Press `Ctrl+,` (or `Cmd+,` on Mac) to open settings
5. Enter your OpenRouter API key
6. Start chatting!

## Keyboard Shortcuts

### General
| Shortcut | Action |
|----------|--------|
| `?` | Show keyboard shortcuts help |
| `Esc` | Close modals / Focus input |
| `Ctrl+K` | Focus input |
| `Ctrl+,` | Open settings |

### Chat
| Shortcut | Action |
|----------|--------|
| `Enter` | Send message |
| `Shift+Enter` | New line in message |
| `Ctrl+L` | Clear current chat |
| `Ctrl+D` | Delete current chat |

### Navigation
| Shortcut | Action |
|----------|--------|
| `Ctrl+N` | Create new chat |
| `Ctrl+H` | Toggle chat history sidebar |
| `Ctrl+/` | Focus search in chat history |
| `Ctrl+1-9` | Switch to chat 1-9 |

## Design Philosophy

Pro-Chat is designed for individual users who prefer keyboard-driven workflows. The interface is clean and distraction-free, inspired by OpenWebUI's design principles but focused on personal use rather than team collaboration. Every feature is accessible without touching the mouse, and the UI stays out of your way while you focus on the conversation.

## Browser Support

Works in all modern browsers that support ES6+ JavaScript and CSS Grid/Flexbox.
